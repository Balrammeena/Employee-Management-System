## Employee Management System (EMS)

A full-stack **Employee Management System** developed using **Django**, **MySQL**, **HTML**, **CSS**, and **JavaScript**.  
This project provides a modern, responsive admin dashboard to manage employee records efficiently with secure authentication and role-based access.



##  Features

-  Secure Login & Logout (Django Authentication)
-  Modern Admin Dashboard UI
-  Employee CRUD Operations (Add, Edit, Delete, View)
-  Department-based Employee Management
-  Employee History Table
-  Export Employee Data (CSV)
-  Fully Responsive (Desktop, Tablet, Mobile)
-  Sidebar Navigation & Topbar Controls
-  Clean UI with reusable components


##  Tech Stack

- Backend: Python, Django
- Frontend: HTML5, CSS3, JavaScript 
- Database: MySQL
- Authentication: Django Auth
